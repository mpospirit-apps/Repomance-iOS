# Repomance iOS

A mobile app for discovering and starring GitHub repositories through a dating app like swipe interface.

## Features

- **Swipe Interface**: Intuitive swipe-based interaction for discovering repositories
- **GitHub Integration**: OAuth authentication and seamless starring of repositories
- **Smart Filtering**: Filter repositories by categories, star count ranges
- **Repository Details**: View comprehensive information including:
  - Repository description and statistics
  - Language breakdown
  - README previews
  - Topics and license information
- **Batch Discovery**: Personalized batches of repositories based on your preferences
- **Statistics**: Check your dismissed and starred repository counts

The app requires GitHub authentication to star repositories. No personal data is collected or stored beyond what's necessary for GitHub API interaction.
